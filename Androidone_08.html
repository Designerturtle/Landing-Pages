What is Android One?
Android One is a new family of Android™ devices, designed in collaboration with several manufacturers. All Android One devices run the latest version of Android, get updates from Google, and come with a hardware package tuned for Android.
Is Android One a new version of Android OS?
No. Android one is family of smartphones running Lollipop 5.1, the latest version of Android
Why should I buy an Android One device?
Android One devices run Lollipop 5.1, the latest and fastest version of Android. You,ll enjoy a smooth, fast, lag-free experience and access to the latest technology, features and apps. All at a really affordable price
